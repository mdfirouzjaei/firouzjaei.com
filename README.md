# وب سایت خانوادگی فیروزجایی

این پروژه یک نسخه اولیه و ایستا برای وب سایت خانوادگی است و برای اجرا روی GitHub Pages با دامنه مدیریت شده در Squarespace مناسب است.

## اجرا

```powershell
python -m http.server 8080
```

سپس نشانی زیر را باز کنید:

```text
http://localhost:8080
```

## ورود آزمایشی مدیر

ایمیل مالک:

```text
mdfirouzjaei@gmail.com
```

رمز عبور آزمایشی:

```text
owner-demo-1403
```

## نکته امنیتی

این نسخه برای نمونه سازی است. چون سایت ایستا است، رمز مدیران داخل مرورگر ذخیره می شود و برای استفاده واقعی امن نیست. برای نسخه نهایی باید احراز هویت و ذخیره داده ها با یک سرویس پشتیبان مثل Supabase، Firebase، Clerk یا یک API اختصاصی انجام شود.
