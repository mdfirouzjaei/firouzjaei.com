# وب سایت خانوادگی فیروزجایی

این پروژه یک نسخه اولیه و ایستا برای وب سایت خانوادگی است و برای اجرا روی GitHub Pages با دامنه مدیریت شده در Squarespace مناسب است.

## اجرا

```powershell
python -m http.server 8080
```

سپس نشانی زیر را باز کنید:

```text
http://localhost:8080
```

## ورود مدیر

ایمیل مالک:

```text
mdfirouzjaei@gmail.com
```

## نکته امنیتی

ورود مدیر و بررسی رمز در Azure Functions انجام می‌شود. رمزها به‌صورت هش‌شده در فضای خصوصی Azure نگه‌داری می‌شوند و نباید داخل مخزن GitHub ثبت شوند.

## پشتیبان کامل

مالک می‌تواند از بخش «داده‌ها» در پنل مدیر، یک فایل JSON شامل داده کامل محتوایی سایت و نسخه درون‌گذاری‌شده همه رسانه‌های ارجاع‌شده دریافت کند. بازیابی، فایل‌ها را با بررسی SHA-256 و به‌صورت جداگانه در Azure ذخیره می‌کند و سپس داده سایت را به‌طور یک‌جا جایگزین می‌کند. حساب‌های مدیران، رمزها و کلیدهای امنیتی در جریان بازیابی تغییر نمی‌کنند.
