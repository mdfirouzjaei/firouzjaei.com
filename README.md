# وب سایت خانوادگی فیروزجایی

این پروژه یک نسخه اولیه و ایستا برای وب سایت خانوادگی است و برای اجرا روی GitHub Pages با دامنه مدیریت شده در Squarespace مناسب است.

## اجرا

```powershell
python -m http.server 8080
```

سپس نشانی زیر را باز کنید:

```text
http://localhost:8080
```

## نکته امنیتی

ورود مدیر و بررسی رمز در Azure Functions انجام می‌شود. رمزها به‌صورت هش‌شده در فضای خصوصی Azure نگه‌داری می‌شوند و نباید داخل مخزن GitHub ثبت شوند.
